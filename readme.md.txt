# 🧳 Agentic AI Travel Assistant

This project demonstrates an agentic AI system built using CrewAI and LLMs.  
It autonomously plans travel — including flights, hotels, and weather checks — using real-time APIs.

## 📂 File

- `Lab2_Part3_AI_Travel_Agent.ipynb`: The core notebook demonstrating the agent flow.

## 🛠️ Technologies

- Python
- CrewAI
- LLMs
- API integration
